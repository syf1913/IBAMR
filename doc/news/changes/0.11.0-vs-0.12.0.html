Incompatibilities

  1. Changed: INSStaggeredHierarchyIntegrator now outputs pressure, velocity, and vorticity data as a nodal, not cell-centered, fields.
    (David Wells, 2022/10/03)
  2. Changed: The function signature of HierarchyMathOps::curl() for side-valued input and nodal output has been incompatibly changed to allow for setting source and destination data synchronization.
    (David Wells, 2022/11/15)
  3. Changed API: Global static system names for IBFEMethod and related classes are deprecated. Methods are now provided by classes FEMechanicsBase, IBFEMethod, and FEDataManager to access system names for key libMesh systems maintained by those classes.
    (Boyce Griffith, 2022/11/16)

Major Changes

  1. New: Implementations of analytically divergence-free interpolation and spreading kernels.
    (Boyce Griffith, 2022/10/27)

Minor Changes

  1. New: Added new functions for interpolating from side- or face-centered data to nodal data.
    (David Wells, 2022/10/03)
  2. Fixed: Several bugs in IBFEInstrumentPanel have been fixed; the class now works correctly.
    (Boyce Griffith, Charles Puelz, David Wells, 2022/10/26)
  3. Improved: A node-centered curl operator is now provided in 3D and used for vorticity output for INSStaggeredHierarchyIntegrator.
    (Boyce Griffith, 2022/10/30)
  4. Fixed: IBFEInstrumentPanel generates perimeter curves in a more robust and consistent way.
    (Boyce Griffith, Charles Puelz, 2022/11/02)
  5. Improved: IBFEInstrumentPanel now reports pressures at the centroids of the meter meshes.
    (Boyce Griffith, David Wells, 2022/11/02)
  6. Improved: Add SPP-RK3 implementations to the explict FE mechanics solver.
    (Boyce Griffith, 2022/11/15)
  7. Improved: Improved support for damping in the explicit FE mechanics solver.
    (Boyce Griffith, 2022/11/17)
  8. New: IBAMR provides an example of how to implement a DLM FSI scheme for elastic structures (in examples/DLM/ex0).
    (Boyce Griffith, David Wells, 2022/11/17)
